# 0 - Intro
This is the start of me documenting my progress with the [SO-101 robot arm](https://github.com/TheRobotStudio/SO-ARM100). hopefully this works 😛


First step was assembly. I ordered the electronics only kit from Partabot and had some friends at the University of Washington print out the parts.
